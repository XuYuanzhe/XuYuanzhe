# Vim Command 常用操作

```
:q      退出
:q!     强制退出，不保存
:w      保存
:wq     保存并退出
:e!     放弃所有修改，重新载入文件


h       左
j       下
k       上
l       右
0       移动到这一行的首字符
$       移动到这一行的末字符
10j     下移10行
10gg    跳转到第10行
G       移动到文件的最后一行
20G     移动到文件的第20行
gg      移动到文件的首行


i       在光标前插入
a       在光标后插入
I       行首插入
A       行末插入


dd      删除本行
ddp     交换上下两行
dw      删除游标所在单词
d$      删除到行尾
dG      删除到文件尾


:set nu         显示行号
:set nonu       取消行号
:set hls        高亮搜索结果

```
